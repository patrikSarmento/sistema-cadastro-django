# Sistema de Cadastro de Clientes com Django

Este projeto e um sistema simples de cadastro de clientes desenvolvido com Django.  
Ele permite gerenciar clientes de forma facil, criando, editando, excluindo e listando os registros em uma interface web basica.

## Funcionalidades

Criar novos clientes  
Editar clientes existentes  
Excluir clientes  
Listar todos os clientes cadastrados  
Sistema baseado em Django com PostgreSQL

## Tecnologias Utilizadas

Python 3.x  
Django 6.x  
PostgreSQL como banco de dados  
HTML e CSS para a interface basica

## Como Rodar o Projeto

1. Clone o repositorio
   git clone https://github.com/patrikSarmento/sistema-cadastro-django.git

2. Entre na pasta do projeto

3. Crie e ative um ambiente virtual

4. Instale as dependencias

5. Configure o banco de dados PostgreSQL no arquivo core/settings.py

6. Rode as migrations

7. Inicie o servidor

8. Acesse o sistema pelo navegador

http://127.0.0.1:8000

## Observacoes

Certifique-se de que o PostgreSQL esta instalado e rodando  
Atualize as credenciais do banco no arquivo settings antes de rodar o projeto  
Este projeto e um ponto de partida, podendo ser expandido com mais funcionalidades
